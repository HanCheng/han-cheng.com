My Projects

Jan 2014

ChatRoom Application

Android App

I use this chatroom android App to send messages with my girlfriend thousands of miles away. She can register and upload an image through camera or albums in the phone.
I designed this app by implementing server and android client using Java. SQLite database is used to store user information and JSON is used to transmit the data between server and client. Also Multi-thread technique is been used to connect the server and multi-users.

Wireless Robot Car with Camera

Final Project, Embedded System

I first proposed this idea, designed and integrated the hardware and the software. The hardware platform includes STM32 board, WIFI router, motor driver board, and camera. They communicate with UART. The software is android app, which sends the control signals to the car and receives the image signals from the wifi router.
